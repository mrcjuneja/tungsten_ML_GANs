# tungsten_ML_GANs
Synthetic microstructure generation for tungsten plasma-facing components for given exposure conditions

This repository contains training codes and scripts used for baseline c-GAN and StyleGAN2-ADA models.

Larger files can be found here: https://drive.google.com/drive/folders/1T05gc5W9UECaM4duflW4BxawMoP22vr1?usp=sharing
This contains model checkpoints.
